Frequently asked questions

I can't see my runs or compare runs?

In order to select runs a log in is required. This can be done by selecting the menu in the top left corner and select log in.

I can't log in?

In order to log in an account has to be created. This can be done by selecting the menu in the top left corner and select register.

All of the information in my runs is shown at the same time?

The graphs can be modified by selecting what is desired to be shown by the buttons on the right side of the graph.

I can't compare runs?

In order to compare runs selection on what is desired to compare has to be done fistly by choosing from the buttons on the right side of each graph. After that select the runs, which are desired to be compared.

I can't remove runs while comparing?

After selecting runs they can be removed by selecting the Remove runs button. There can be seen all the selected runs and by pressing them they can be removed
